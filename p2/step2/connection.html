All of these examples of menus investigate relationships between menu icons and where they use them.
In the Universal page, it shows examples of general menu icons. People use their own logo, home figures and hamburger icon for their menu. It is something everyone recognizes and easy to navigate.
The Transformation page shows how hamburger icon change, it gives information or sometime just aesthetics.
The Web page shows that there is not always a menu button because web pages have a lot of space. People use their own logo or site name.
For the Mobile page, there is limited space on mobile devices compared with web pages. Mobile devices use universal menu icons, which everyone can recognize.
All of these relationships of the menu icon depend on what platforms are being used.
